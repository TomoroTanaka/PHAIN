**Dataset**

This folder is to contain datasets.

We have already made the folder 'Examples', which contains 4 data from "LibriSpeech".
This corpus is freely available under CC BY 4.0 license (https://creativecommons.org/licenses/by/4.0/deed.en).
For more information about this corpus, visit 'https://www.openslr.org/12/' and see the paper 
"LibriSpeech: an ASR corpus based on public domain audio books", Vassil Panayotov, Guoguo Chen, Daniel Povey and Sanjeev Khudanpur, ICASSP 2015.

You can substitute any other files for them. If so, you need to modify the main file a little bit.
